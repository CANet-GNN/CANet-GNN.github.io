Abstract

Graph Neural Networks (GNNs) have recently gained tremendous popularity as they showed promising results in solving graph-structure data that are very common in real-world applications. However, most models can only handle local interactions in a graph using the message-passing paradigm (i.e., interaction within a neighborhood of a few nodes). When we enforce the models to use information from far away nodes, we will encounter two major issuesâ€”over-smoothing \& over-squashing. To solve this, architectures such as transformer and diffusion models are introduced. Yet, these models are not tested on large graph datasets with long-range interactions. Hence, this paper will validate existing state-of-the-art (SOTA) models on a specifically designed long-range interaction dataset.

Discussion

There still exists a lot of room for improvement.
- We can use a more sophisticated clustering algorithm to cluster the nodes.
- We can use a more sophisticated graph neural network to learn the node embeddings.
- We can use a more sophisticated graph neural network to learn the cluster embeddings.
- We can use a more sophisticated positional encoder to encode the node positions.
- We can use an efficient attention mechanism to compute the attention weights for increasing cluster number and hierachy for larger graphs.
